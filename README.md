# 🐍 Snake Game

A classic **Snake Game** developed in **Java** using the **Swing** framework. The game allows players to control a snake, collect food to increase their score, and avoid collisions with the walls or the snake's own body.

---

## 📌 Features

- 🎮 Smooth keyboard controls
- 🍎 Random food generation
- 📈 Real-time score tracking
- 💥 Collision detection
- 🔄 Restart game after Game Over
- 🖥️ Simple and responsive GUI using Java Swing

---

## 🛠️ Technologies Used

- Java
- Java Swing
- AWT (Abstract Window Toolkit)

## 🚀 Getting Started

### Prerequisites

- Java JDK 8 or above
- IntelliJ IDEA / Eclipse / VS Code

### Installation

1. Clone the repository

```bash
git clone https://github.com/krashan97/Snake-Game.git
```

2. Open the project in your preferred IDE.

3. Compile and run the main Java file.
